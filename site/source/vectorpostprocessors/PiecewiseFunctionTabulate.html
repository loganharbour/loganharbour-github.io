      PiecewiseFunctionTabulate

      Tabulate the function nodes of a piecewise function, such as PiecewiseLinear or PiecewiseConstant

      This object can be used to check the function generated by a 2D piecewise defined function, such as

      • PiecewiseConstant

      • PiecewiseLinear

      • CoarsenedPiecewiseLinear.

      Input Parameters

      • functionName of the piecewise function object to extract the time steps from

        C++ Type:FunctionName

        Options:

        Description:Name of the piecewise function object to extract the time steps from
